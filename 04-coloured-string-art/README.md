## Coloured String Art

Extension of the string art concept to support full-color image generation.

The project focuses on developing methods for translating images into multi-layered threading instructions using a chosen color palette that best represents the image.

Due to its experimental and ongoing nature, the implementation details are not publicly available.

This project is currently under active development, with the goal of significantly improving visual quality and efficiency.

Future plans include designing and building a dedicated machine capable of automatically creating full-color string art pieces.
