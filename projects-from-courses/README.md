## What is Boot.dev?

[Boot.dev](https://boot.dev) is an online learning platform focused on backend development and computer science fundamentals.

It provides hands-on, practice-oriented learning through short lessons combined with coding exercises.  
The platform emphasizes learning by doing — each concept is followed by practical implementation tasks.

One of its key features is the challenge system, which regularly reintroduces problems from previous topics.  
This helps reinforce knowledge, improve retention, and build problem-solving skills over time.

### Topics I covered:
- Object-Oriented Programming (encapsulation, inheritance, polymorphism)
- Functional Programming (pure functions, closures, decorators, recursion)
- Data Structures & Algorithms (Big O analysis, implementing data structures and sorting algorithms from scratch)
- Linux fundamentals
- Git and version control

## What is freeCodeCamp?

[freeCodeCamp](https://www.freecodecamp.org) is a widely recognized platform offering structured programming courses with a strong focus on practical projects.

Courses are built around real-world tasks and require completing projects to pass each section.  
This approach ensures not only understanding of theory, but also the ability to apply it in practice.

The projects included in this repository were completed as part of the **Scientific Computing with Python** certification.
