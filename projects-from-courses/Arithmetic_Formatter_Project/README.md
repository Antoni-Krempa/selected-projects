# Arithmetic Formatter

Project completed as part of the **freeCodeCamp – Scientific Computing with Python** course.

## Description

The goal of this project is to format a list of arithmetic problems so that they are displayed **vertically and side by side**, similar to how they are written in elementary school.

The program supports addition and subtraction operations and can optionally display the **calculated results**.

## Supported operations
- Addition (`+`)
- Subtraction (`-`)

## Example

```python
arithmetic_arranger(["32 + 698", "3801 - 2", "45 + 43", "123 + 49"], True)

Wynik:

+  8    - 3801    + 9999    -  49
